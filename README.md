# Lytup-CLI
CLI for Lytup.comv - A blazing file transfer platform

## Installation
```sh
$ npm i -g lytup
```

## Usage
### Upload files
###### CLI
```sh
$ lytup -u <files>
```
![CLI-upload](http://d.pr/i/XVIM+)
###### Browser
```
http://lytup.com
```
![Browser-upload](http://d.pr/i/dsW3+)

### Download files
###### CLI
```sh
lytup -d <code>
```
