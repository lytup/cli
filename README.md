# Lytup-CLI #
CLI for Lytup.com

## Installation
```sh
$ npm i lytup -g
```

## Usage

### Upload file(s)
```sh
$ lytup -u chromecast-setup.dmg node-v0.10.15.pkg
download url => http://lytup.com/kB7L5
transferring /  |  27%  |  695 KB/s
```


### Download file(s)
```sh
$ lytup -d http://lytup.com/kB7L5
transferring |  |  98%  |  694 KB/s
```
