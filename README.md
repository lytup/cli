# Lytup-CLI
Command-line interface for [Lytup](http://lytup.com) - A blazing fast file transfer platform

## Installation
```sh
$ npm i -g lytup
```

## Usage

### Upload files
```sh
$ lytup <files>
```
![CLI-upload](http://d.pr/i/XVIM+)

### Download files
```sh
lytup -d <code>
```
